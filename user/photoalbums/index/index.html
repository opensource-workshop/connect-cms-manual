    フォトアルバム - アルバム表示

    アルバムの一覧や表紙に追加した写真・動画の一覧が表示されます。

    画面

    【写真、動画の一覧】

    • アルバムを作成し、写真や動画を整理できます。
    • 表示設定することで、アルバムごとのダウンロード機能を表示することもできます。

    【写真の一覧】

    • 写真はこのように一覧で表示されます。

    【写真の拡大表示】

    • 写真をクリックすると、このように拡大表示されます。

    【動画の表示】

    • 動画は再生、最大化再生などが可能です。表示設定で埋め込みコードを表示するにしている場合は、埋め込みコードも表示されます。
